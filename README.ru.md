# conformal-nutrition

Мультимодальная регрессия нутриентов блюда с конформной калибровкой предсказательных интервалов.

Магистерская работа. По фотографии блюда и списку ингредиентов модель оценивает калории и макронутриенты — жиры, углеводы, белок — и выдает откалиброванные интервалы через CQR (conformalized quantile regression).

[English version](README.md)

## Подход

- Визуальный энкодер: DINOv2-small, заморожен, вход 224×224 с нормализацией ImageNet.
- Текстовый энкодер: sentence-transformers MiniLM-L6, заморожен, mean-pooling.
- Слияние модальностей: gated-комбинация визуального и текстового эмбеддингов.
- Регрессионная голова: многослойный перцептрон с выходами на квантили 0.05, 0.50, 0.95.
- Конформная калибровка: CQR (Romano, Patterson, Candès, NeurIPS 2019) и split conformal prediction как baseline.

## Данные

[Nutrition5k](https://github.com/google-research-datasets/Nutrition5k), Thames et al., CVPR 2021. Используется overhead RGB подмассив — около 3 200 блюд с измеренными значениями массы, калорий и нутриентов.

## Структура репозитория

```
notebooks/   ноутбуки пайплайна 01–05 (подготовка данных, визуальный baseline, слияние, калибровка, бенчмарк)
src/         общие утилиты (постепенно наполняется; основной код экспериментов — в ноутбуках)
scripts/     скрипт конвертации в CoreML для запуска на устройстве
docs/        текст работы и иллюстрации
```

## Воспроизведение экспериментов

Все эксперименты запускаются на Kaggle. Каждый ноутбук публикует результаты как Kaggle Dataset, который подключается следующим ноутбуком на вход.

| Ноутбук | Выходной датасет                            |
|---------|---------------------------------------------|
| 01      | `dbkarashev/nutrition5k-overhead-rgb-224`   |
| 02      | `dbkarashev/nutrition5k-visual-baseline`    |
| 03      | `dbkarashev/nutrition5k-multimodal-fusion`  |
| 04      | `dbkarashev/nutrition5k-conformal-intervals`|
| 05      | `dbkarashev/nutrition5k-iphone-microbench`  |
